# Tuktuki: LFR Prototype for Tertiary Service

**Tuktuki** is a Line Following Robot (LFR) prototype developed as part of coursework covering digital logic circuits and embedded systems. The project demonstrates real-time sensor-based navigation and control using a microcontroller-driven embedded system.

---

## Course Information

| Detail | Description |
|--------|--------------|
| **Course** | Digital & Logic Circuits, and Microprocessor and Embedded Systems |
| **Microcontroller** | ESP32 WROOM |
| **Coding Environment** | Arduino IDE |

---

## Project Files

| File | Description |
|------|--------------|
| `LFR.ino` | Core line-following logic and motor control |
| `AI-Module.ino` | AI/decision-making module for the robot |
